## 当システムの使い方
前提条件：使用するPCにDockerがインストールされていること。<br>
インストールされていない場合は、[こちら](https://www.docker.com/ja-jp/products/docker-desktop/)からインストールをする。<br>
<br>
1\. Dockerを起動する。<br>
2\. お使いのOSに合わせて、以下の手順でファイルを起動。<br>
    Macの場合<br>
    shフォルダのsetUpSystem.shを起動する。

```
sh setUpSystem.sh
```
